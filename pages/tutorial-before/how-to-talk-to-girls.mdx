# 如何与女生聊天

这应该是最难的部分了吧。

## 首要前提

首先是，做自己。

不要为了迎合女生，故意改变自己。

比如说，对方说她喜欢听 BTS，但是你本身对 BTS 并不感兴趣，那么就不要闲着没事，告诉她说你也喜欢 BTS。这样做有好几个后果：

1. 她因此对你感兴趣，然后经常和你聊起这个，你到时候会很尴尬，逃不出这个圈，即时谈上了，也终究会支离破碎。
2. 过度的迎合会让女生丧失兴趣。
3. 你丧失了自我，变成🤡

所以，你就是你，做自己，变得自信才是最重要的。

时刻记住这一个，这是所有事情的前提。

## 网络上

和女生聊天很复杂，而且不同女生，也需要有不同的应对方法。

但是，有一个通用的原则你需要遵守，这样，对付大部分女生，基本都管用了。

在你们聊天中，你作为一个男生，千万不能表现出你很渴求她，以及你对她的期望以及不确定。

怎么理解呢？我们拿一个实际案例进行分析：

比如，当你想约她出去图书馆的时候，你这样说（反面教材）

> **你**：对了，你周六有空吗（）（）
>
> **她**：有，怎么了
>
> **你**：我也有空，正好要期末了，你看我们要不要一起（）去图书馆复习（（（
>
> **她**：呃，我看看吧
>
> **你**：好的好的（），决定了就告诉我
>
> [END]

这么短短的一段聊天，我们可以总结出好几个错误：

1. 你使用标点符号的次数太多，给人印象过于紧张和不自信。
2. 你没有主权，让她占据了话语上的优势。
3. 你说话没有风度，没有给自己保留神秘感。
4. 在对方表现出不感兴趣的时候，继续舔

接下来我们逐个分析。

首先是口头禅的问题比如 `（）` 以及多个 `（（（`。带这个东西，有些女生就会觉得这个男生好奇怪。所以，你要做的事尽量让你发的消息不带任何口头禅。同时，尽量让自己的信息简短精炼，不要给她发小作文。

追女生，最重要的就是你俩的关系的平衡。

嘴上说不好表达，想象一个天平。

如果你苦苦央求女生，那么这个时候你就在低位，女生在高位，她有主权，你只能服从，被拒绝。

但是如果你变得 casual，随便一点，然后（装作）不在乎对方回应的时候，你就在高位，女生在低位，此时，你有更多的选择权以及不会因为女生的拒绝而伤心。

请记住这个一个感情低位的原则。

所以，当你说

>  你看我们要不要一起去图书馆复习

时，你就把主权交给了女生，你让她同时决定了 “你” 和 “她” 去不去图书馆。

相反，这时候正确的做法是说 

> 我周末准备去图书馆复习，你来吗

这个时候，你表明了你已经有计划，自己去图书馆，女生的参与是可选的，附带条件。这个时候，你就拿了上风，主导了整个事件。

然后，非必要信息，不要在网络聊天上透露。比如你的 "我也有空"。当对方没有问起这个的时候，你不需要告诉她，保留一定的神秘感，降低你表现出的需求感。如果你从头就告诉她关于你的所有事情，她有可能只会对你更加地厌烦。相反，这些事情都应该等你们见面以后，再聊起来。

最后，就是当对方不感兴趣时，你的反应。

这个时候，常规做法是简短地，回一句。可以是 “ok" 或者 “好的“。

但是，不要发很长的消息，然后透露出你的需求感。你要做的，就是冷漠。因为对方已经对你不感兴趣了，甚至可以直接不回。

综上所述，和女生在网络上聊天，可以总结出以下几点：

1. 摈弃标点和口头禅，少发长消息
2. 随意一点，展现出你的风度和话语权，以及你的自主决策的能力
3. 保留神秘感，见面再说
4. 在对方表现出不感兴趣的时候，回复得简短，冷静，不要追问，不要暴露你的需求感
